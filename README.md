## Description
   
Zookeeper is a catalog management system that allows users to search for animals or zookeepers within the database. 

    - Built with: 
        * JavaScript
        * Node JS
        * Express JS
        
<p>&nbsp;</p>

## Usage

    - Install:
        * npm install
        
    - Run:
        * npm start
        * go to http://localhost:3000/
